# NetWatch: Advanced Network Discovery & Security Assessment Tool

## Overview
NetWatch is a sophisticated network scanning and visualization platform designed to provide comprehensive local network intelligence and security insights. Built as a professional portfolio project, this tool demonstrates advanced cybersecurity skills through detailed network device discovery, classification, and risk profiling.

## Key Features
- **Comprehensive Network Mapping**: Utilize advanced scanning techniques to discover and catalog network devices
- **Device Classification**: Intelligent identification and categorization of network endpoints (IoT, CCTV, laptops, mobile devices)
- **Security Risk Assessment**: Generate risk scores and identify potential vulnerabilities
- **Interactive Web Dashboard**: Dark cyberpunk-themed interface with real-time network topology visualization
- **Export Capabilities**: JSON and CSV export for further analysis
- **REST API Integration**: Programmatic scan control and monitoring

## Technical Stack
- Network Scanning: `arp-scan`, `nmap`
- Web Framework: Flask
- Frontend: Dark cyberpunk design
- Device Identification: MAC Address OUI Lookup
- Platform: Linux-based environments

## Ethical Considerations
This tool is developed strictly for educational and professional demonstration purposes, adhering to responsible cybersecurity practices and emphasizing authorized network scanning.

## Purpose
Created as a professional portfolio project to showcase network security skills, demonstrating technical proficiency in network discovery, device intelligence, and security assessment techniques.
